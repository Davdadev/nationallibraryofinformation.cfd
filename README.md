# National Library of Information

A complete static archive website ready for GitHub Pages.

## What is included
- 20+ finished pages
- Home, discover, research, learn, news, blog, events, exhibitions, archives
- Search page with live filtering
- 10 fully written article pages
- Collection detail pages
- Responsive layout
- GitHub Pages-compatible hash routing

## Run locally
Open `index.html` in a browser or serve the folder with a static server.

## Publish to GitHub Pages
1. Create a new repository.
2. Upload all files in this folder.
3. Enable GitHub Pages from the root branch.
4. Your site will work without a build step.
